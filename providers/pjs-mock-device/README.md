# pjs-mock-device

A PJs device adapter which can be used for testing

### Dependency coordinates for Maven

```xml
<dependency>
    <groupId>io.github.iamnicknack</groupId>
    <artifactId>pjs-mock-device</artifactId>
    <version>0.1.0</version>
</dependency>
```

### Dependency coordinates for Gradle

```kotlin
implementation("io.github.iamnicknack:pjs-mock-device:0.1.0")
```
