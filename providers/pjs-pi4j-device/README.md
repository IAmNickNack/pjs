# pjs-pi4j-device

A PJs device adapter which uses Pi4J to communicate with hardware.

### Dependency coordinates for Maven

```xml
<dependency>
    <groupId>io.github.iamnicknack</groupId>
    <artifactId>pjs-pi4j-device</artifactId>
    <version>0.1.0</version>
</dependency>
```

### Dependency coordinates for Gradle

```kotlin
implementation("io.github.iamnicknack:pjs-pi4j-device:0.1.0")
```
